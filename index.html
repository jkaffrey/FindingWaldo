Where is Waldo?

Instructions

In the script tag below update the function findWaldo to:

  • Search the document for an element that contains the text, 'Waldo'
  • When it finds the tag add a style to make it visible.
  • Finally, return the element ID.
  • You cannot use jquery, getElement* or querySelector* methods
  • Instead you have to use the following methods:
    • childNodes
    • firstChild
    • lastChild
    • parentNode
    • nextSibling
    • previousSibling
